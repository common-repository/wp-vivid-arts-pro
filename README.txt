Artist directory and profile integration.

== Description ==

Are you looking for a modern, clean and simple content management system for your users?  Vivid Arts Pro is a page template integration that display your user directory, their profiles and more.

Vivid Arts Pro is focused on ease of integration, it’s simplicity and it’s modern design.  Built for our internal CMS, Vivid Arts Pro has the admin and the creatives in mind to make the user experience simple and enjoyable.

Enable registered members have the ability to create and manage profiles, or administer a community of user profiles.

== Installation ==

Installation Instructions

From your WordPress dashboard

-Visit ‘Plugins > Add New’
-Search for ‘Vivid Arts Pro’
-Activate Vivid Arts Pro from your Plugins page. (You will be greeted with a Welcome page.)

From WordPress.org

1. Upload `wp-vivid-arts-pro.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Enterprise Id on `/wp-admin/admin.php?page=vivid-arts-pro`
